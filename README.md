# 프로젝트: AI팝과 메신저 롤백 이슈에서 출발한 Python 기반 메신저

2025년 11월 6일 어느냘. 저는 점심시간에 유튜브를 보고 있었습니다. AI로 누구나 쉽게 음악을 만들 수 있게 되면서, AI로 만든 모 메신저 회사를 비판하는 노래와 뮤직비디오를 우연히 보게 되었습니다.

이 "모 메신저팝"은 메신저 업데이트를 꼬집는 내용을 담아 약 140만 명의 호응을 받았습니다.

영상속 내용중 뜨거운 감자로 떠오른 모 메신저의 "롤백 불가" 정책을 직접적으로 공감하며, 

결국 나 이대현이 스스로 메신저를 만들어 보면 어떨까? 라는 생각의 틀로 출발하게 됐습니다.

제목  : 바로 메신저 앱 정상화~
---
## 기능
- [ ] 방 개설(Create Room): 존재하지 않으면 생성, 있으면 재사용
- [ ] 방 선택(Join Room): 방 목록 조회 또는 이름으로 바로 접속
- [ ] 텍스트 채팅: WebSocket 기반 실시간 송수신(닉네임 표시)

---
## 기술 스택
- 언어: Python 3.11+
- 서버: FastAPI(WebSocket)
- 클라이언트: 선택 ① PyQt6 간단 UI, ② 터미널 CLI 클라이언트(websockets)
- 데이터: In‑Memory 저장(개발 편의). 필요 시 SQLite로 교체 가능
