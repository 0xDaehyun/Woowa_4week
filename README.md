# [BE] 이대현 우테코 오픈미션 프로젝트: AI팝과 메신저 롤백 이슈에서 출발한 Python 기반 메신저

2025년 11월 6일 어느냘. 저는 점심시간에 유튜브를 보고 있었습니다. AI로 누구나 쉽게 음악을 만들 수 있게 되면서, AI로 만든 모 메신저 회사를 비판하는 노래와 뮤직비디오를 우연히 보게 되었습니다.

이 "모 메신저팝"은 메신저 업데이트를 꼬집는 내용을 담아 약 140만 명의 호응을 받았습니다.

영상속 내용중 뜨거운 감자로 떠오른 모 메신저의 "롤백 불가" 정책을 직접적으로 공감하며, 

결국 나 이대현이 스스로 메신저를 만들어 보면 어떨까? 라는 생각의 틀로 출발하게 됐습니다.

제목  : 바로 메신저 앱 정상화~
---
## 기능
- [o] 텍스트 채팅: WebSocket 기반 실시간 송수신
- [o] 닉네임 커스텀
---
## 사용 방법
1. 먼저 Server.py 를 통해 서버를 엽니다. 
2. GUI.py를 통해 채팅창을 열 수 있습니다. 
---
## 주의사항! 
- 이 프로젝트는 교육용으로 구현된 간단한 소켓 기반 메신저입니다.
- 기본 설정에서는 `127.0.0.1`(localhost)를 기준으로 통신하기 때문에 **같은 컴퓨터 안에서만** 채팅이 가능합니다.
- 같은 와이파이(동일 네트워크)에서 다른 컴퓨터를 서버로 사용하려면, 클라이언트의 `HOST` 값을 서버 컴퓨터의 IP(예: `192.168.x.x`)로 직접 바꿔줘야 합니다.
- 공유기 밖(인터넷 외부)에서 접속하려면 포트 포워딩, 방화벽 설정 등 추가 네트워크 설정이 필요하며, 이번 미션에서는 **범위 밖**으로 두었습니다.
